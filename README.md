✏️ Projeto em Destaque: ChikGuard PRO

O Futuro da Avicultura Inteligente

O ChikGuard é um sistema de monitorização térmica e comportamental que integra hardware e software para reduzir a mortalidade em granjas.

Cérebro: Backend em Python com processamento de imagem em tempo real.

Visão: IA treinada com YOLOv8 para contagem de aves e deteção de intrusos.

Interface: Dashboard em React (Web) e App nativa (React Native) para controlo remoto.

Edge: Implementação em Raspberry Pi com acesso remoto via Cloudflare Tunnels.

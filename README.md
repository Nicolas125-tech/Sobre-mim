### 🌟 Projeto em Destaque: ChikGuard

O **ChikGuard** é um sistema SaaS AgTech que utiliza visão computacional e Inteligência Artificial na borda (*Edge Computing*) para monitorar a saúde e o comportamento de aves em granjas.

**Principais Desafios Técnicos Superados:**
* **IA na Borda (Edge):** Implementação e otimização do modelo YOLOv8 com OpenCV rodando em hardware limitado (Mini PC) com técnicas de *Tracking* (BoT-SORT) e *Frame Skipping* para maximizar FPS e precisão.
* **Resiliência e IoT:** Arquitetura desacoplada e contentorizada com Docker (restart: always) e rotinas de *auto-reconnect* de câmeras para tolerância a falhas físicas e de rede.
* **Segurança Robusta (Supabase):** Implementação de IAM e RBAC Multi-Tenant com políticas estritas de Row Level Security (RLS) diretamente na base de dados PostgreSQL.
